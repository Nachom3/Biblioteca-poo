Sistema de Gestión de Biblioteca para una biblioteca municipal.

## Configuración de la base de datos

La ruta de la base de datos está hardcodeada en `Data/BibliotecaDbContext.cs`. Antes de ejecutar la aplicación, abrir ese archivo y modificar la ruta del `Data Source` para que apunte a la ubicación del archivo `biblioteca.db` en la ruta correcta. NO VA A FUNCIONAR SI NO.
